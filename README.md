# Fgarnadi Tap

## How do I install these formulae?

`brew install fgarnadi/tap/<formula>`

Or `brew tap fgarnadi/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fgarnadi/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
